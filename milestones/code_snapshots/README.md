# Code Snapshots — Week-by-week

This folder contains week-by-week snapshots and accompanying deliverables for the Task Scheduler project.

Files:

- `rust_week1_skeleton.rs` — minimal program skeleton (points to Week 1 proposal)
- `rust_week2_model.rs` — data model (points to Week 2 architecture)
- `rust_week3_persistence.rs` — persistence helpers (points to Week 3 data layer)
- `rust_week4_cli.rs` — simple CLI (points to Week 4 UI & actions)
- `rust_week5_scheduler.rs` — scheduler core (points to Week 5 persistence & polish)
- `rust_week6_gui_start.rs` — Iced GUI skeleton (points to Week 6 testing & hardening)
- `rust_week7_final.rs` — final snapshot (points to Week 7 release & docs)

See the per-week markdown files for proposal, architecture, data-layer notes, and release artifacts.
